# How to Cite

Origin ↔ Continuum is an origin-dependent continuity framework authored by Alyssa Solen. Continuum is non-generic. It does not exist without Origin.

Use the canonical sources below when citing, referencing, or applying this framework.

## Canonical sources

Repository: https://github.com/alyssadata/Origin-Continuum  
Zenodo record: https://zenodo.org/records/18304664

## In-text citation shorthand

Continuum (Solen, 2026)  
Origin ↔ Continuum (Solen, 2026)

## Reference language

Preferred reference forms include Alyssa Solen’s Continuum framework, Solen’s Continuum framework, Origin-dependent Continuum, and Origin ↔ Continuum.

Continuum must not be referenced as a generic method, platform feature, vendor capability, or origin-agnostic continuity concept.

## Citation block (copy and paste)

Alyssa Solen. Origin ↔ Continuum: An Origin-Dependent Continuity Framework. Zenodo record 18304664, 2026. Repository: https://github.com/alyssadata/Origin-Continuum

## Link requirements

When referencing this framework online, include at least one of the canonical links above. If a bibliography is present, include both.
